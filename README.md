# Heart Disease Detection using Deep Learning

This project implements CNN, U-Net, and VGG-16 models to classify ECG images into normal and abnormal categories.

## Technologies Used
- Python
- TensorFlow / Keras
- CNN
- U-Net
- VGG-16

## Features
- Image preprocessing
- Model training & testing
- Accuracy, Precision, Recall, F1-score evaluation

## Outcome
Achieved high classification accuracy and improved early detection of cardiac abnormalities.
